# 에스코어 AI Code Assistant

### 소개

- 기업용 SW 개발 및 엔지니어링 특화 AI Coding Agent
- 최신 모델 및 오픈소스를 기업 내부에서 안전하게 사용할 수 있도록 인프라까지 패키징하여 구축

### 벤치마크 대상

- CLI / TUI 형 (터미널)
    - OpenCode, OpenClo 등
- 플러그인 / 확장 프로그램 형 (IDE 내 삽입)

- - Continue, Cline 등
- 독립 IDE 형 (IDE 자체가 AI)
    - Cursor, Windsurf 등

### 타겟

- 금융 · 공공 ·제조 업종을 주요 고객으로
- 엔드유저는 기업 내부의 **개발자** 

### 고객 니즈

- SW 개발에 AI 활용하고 싶지만 보안상 코드를 밖으로 보낼 수 없어
- 우리 회사의 오래된 레거시 코드와 내부 개발 표준을 잘 아는 AI가 필요해
- 우리 프로젝트의 파일 구조와 종속성을 다 파악해서 바로 돌아가는 코드가 필요해

### 핵심 소구점

**"Context FrameWork" & "측정 가능한 성과"**

- 기업 고유의 Context
    - 레거시 코드 로직, 사내 표준 프레임 워크, 도메인 특화 용어의 이해
- 보안 Context
    - 사내 보안 규칙을 실시간 맥락으로 주입, 사후 보안 검증이 아닌 작성 시점의 가드레일로 기능
- Engineering Context  
    - 요구사항(기획)부터 배포(Ops)까지의 전체 작업 흐름(Workflow)을 이해
- Manage Agent
    - 개발 데이터 수집을 통한 도입 성과 및 ROI의 정량적 증명

### 구성

- 에이전트 레이어: 개발 전 주기를 전문 에이전트(Plan, Code, Test, Release, Monitor)가 분담

- 컨텍스트 프레임워크: 기업 내부의 코드 자산, 보안 규정, 엔지니어링 표준, 도메인 지식을 결합
- 컨설팅 / 진단: 현황 진단, 도입 전략 수립, PoC 수행 및 조직 변화 관리 및 확산 지원
- 데이터 기반 성과 측정: AI 도입 생산성 지표와 ROI를 대시보드로 제공(Manage Agent)
- 인프라 및 서버: On-Premise LLM 환경을 구축에 최적화된 HW 구성과 SOTA 모델 제공

### 제공 형태

- 모델 라인업
    - On-Premise 환경에서 쓸 수 있는 SOTA 모델 전부
        
- 배포
    - On-Premise 전용

- 운영
    - 동시 접속량 기반 최적 HW 구성 및 로컬 LLM 운영/보안 거버넌스, 폐쇄망 내 LLMOps 제공

###   
오퍼링/가격 체계

- 하드웨어 인프라와 소프트웨어를 통합 패키징하여 구축
- 고객사별 구축 리소스 산정 기반 계약 진행
