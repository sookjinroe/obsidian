        2. 추출된 시나리오를 바탕으로 테스트 에이전트가 단위 테스트 코드를 자동 생성하여 코드 커버리지 70% 이상 확보

        1. 윈도우, 모바일 등 크로스 플랫폼 테스트
        2. AI Agent를 활용하여 스크립트 작성 시간을 대폭 단축하고 테스트 유지보수성 향상
### SK케미칼 품질포탈
- 외부 개발 수행사 개발 프로젝트 너무 많고 본사에는 pm 조직만 있음. 피엠 1명이 열개 이상의 프로젝트 관리
- **1. 문제**

개발 활동 기록은 이슈 트래커, 코드 저장소, 보안 스캐너, CI/CD 파이프라인에 각각 저장됨.  
각 도구는 자기 영역의 데이터만 보여주며 연결되지 않은 데이터는 현황이 아니라 수치  
현황을 파악하려면 여러 도구를 열고, 데이터를 취합하고, 직접 해석필요  
마감일 지연, 생산성 저하 등에 영향

  

**2. 해결**  
품질 포탈은 소프트웨어 개발 주기에서에서 발생하는 수많은 파편화된 시그널을 통합하여, 개발 주기의 예측 가능성과 운영 효율성을 높인다  
품질 포탈은 개발 활동 데이터를 연결하고 AI로 관계를 해석해서 무엇이 잘못되었는지 파악

- 모든 개발 파이프라인의 실시간 진행 상태를 지표화해서 가시성 확보할 수 있고
    - 서로 다른 도구의 데이터를 작업 단위로 묶어주는
- 도메인 간 교차 분석으로 단일 도구로는 보이지 않는 것을 발견할 수 있음
    - 에이전트는 별도의 지시 없이 지속적으로 실행되어 패턴이 나타나는 즉시 위험, 병목 현상, 요구 사항 격차 등을 파악해서 보고하는
- 사용자 친화적인 인터페이스
    - 대시보드 조망 / 질의로 탐색 / 자동 리포트 수신할 수 있는

### 사내 경영정보/재무관리 에이전트
- 경영 정보 Agent: 부서별/기간별 매출, 영업 이익 등 SAP 기반 경영 정보와 mySales 기반 사업기회 정보를 자연어로 조회 및 시각화 할 수 있는 에이전트
- 재무 관리  Agent: 보조부 (SAP) 기반으로 재무제표상의 항목들을 조회 및 요약, 전월과 비교해줘서 재무제표 현황 파일의 비고란 작성을 효율화 해주는 에이전트


### 삼성전자 DX 과제
- 시스템에 800개 db가 있고 컬럼은 800만개, 컬럼 정리가 안되어있어서 ai ready데이터로 만들기 위해 필드 디스크립션 등 정보가 많은 코드로부터 거꾸로 정리하는 작업
- 삼성전자 전사시스템의 컬럼 설명을 생성해야 한다.
- 사전에 정의한 91개 표준항목과 매칭 시켜야 한다.
- 대상/목적/포맷 으로 분리하여 생성한 결과를 조합한다.
- 결과는 한글과 영문 2가지의 버전으로 생성한다.
