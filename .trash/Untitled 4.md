**(2)** **중기전략 추진 현황 및 업데이트**
1) 추진 현황
	**□ Key Account 대상 SW개발 Agent 소개 및 사업 발굴 결과**
	   - 고객사별 추진 현황
	    ∙ 농협정보시스템: 대표이사 Agenda인 'AI 활용 업무 효율화'와 연계하여 
	      적극적인 도입 의사 확인 및 구체적 일정 조율 중
	    ∙ 현대차증권: 커니코리아 컨설팅 완료 후, 현재 스캘터랩스(Skelter Labs) 
	      주관 PoC 진행 중. 향후 본 사업 발주 시 경쟁 입찰 및 제안 참여 준비 필요
	    ∙ 신한EZ손보: 2주간의 PoC 수행 완료 및 도입 의사 확인. 단, 현 기능 수준 
	      대비 비용 효율성(ROI) 부족 판단. 고객사 요청 개선 포인트 보완 및 기술 
	      대응 진행 중
	   - 주요 시사점
	     ∙ 고객 맞춤형 'Value Proposition' 강화: 단순 기능 구현을 넘어 비용 대비 
	      효과(ROI)에 대한 엄격한 검증이 필요. 
	    ∙ 'Top-Down' 영업 기회 확대 및 선점: 경영진의 AI 전환(DX) 아젠다와 
	      연계한 전략적 제안이 유효할 것으로 판단
	    ∙ 컨설팅과의 전략적 연계 통한 '본 사업' 선점: 컨설팅 단계부터 우리 
	      솔루션이 핵심 아키텍처로 반영되도록 '업스트림(Up-stream)' 영업 필요
 2) 중기전략 업데이트
	**□ AX 컨설팅 연계 및 AI Agent 구축 사업 추진**
	   - 사업 구조 전환: 컨설팅 연계형 "AX 완결형" 오퍼링 체계 구축 
	    ∙ 최근 AI 시장은 단순 구축이 아닌 ‘유즈케이스 발굴’중심의 컨설팅社가
	      사업 전체를 주도하는 양상으로, 구축 단독 진입 시 사업 기회 확보 어려움
	    ∙ 에스코어 컨설팅 부문 및 AX TF와의 긴밀한 협업 모델을 구축하여 초기
	      기획 단계부터 참여하여 컨설팅-설계-구축으로 이어지는 사업권 선점 및 수주 경쟁력 강화
	    ∙ 플랫폼 제공 위주에서 탈피하여, 고객의 실질적 Pain 포인트를 해결하는 
	      'AI Agent' 중심의 최종 결과물 제공으로 서비스 지향적 사업 구조 전환
	   - 기술 경쟁력 고도화: AI Agent 개발 실행력 확보
	    ∙ 범용 AI를 넘어 특정 업무에 특화된 Vertical Agent가 핵심 경쟁력이 됨에 
	      따라, 신속한 Agent 구축 능력과 데이터 연결의 정확도가 사업 성패를 결정
	    ∙ 데이터 연결 기술의 고도화를 위해 Text-to-SQL 성능을 개선하고, 비용 
	      효율적인 Semantic Layer 구축 방안 마련
	    ∙ SW 개발 Agent의 핵심 타겟을 ‘AI Agent 자체 개발’ 영역으로 집중하여, Agent 개발 생산성 제고
	
	**□ AI 도구를 활용한 개발 공정 자동화**
	   - 개발 업무 전체 31개 中 7개 항목에 대해 30% 절감, 전체 12% 절감 목표
	     ① 요구사항 표준화: 요구사항 기반 표준화된 SRS 생성
	     ② API 설계: SRS, API 설계 표준 활용 API Spec 생성
	     ③ DB 설계: DB 스키마 및 CRUD 코드 자동 생성
	     ④ 소스코드 생성 고도화: 설계 문서 기반 코드 생성
	     ⑤ 단위 테스트 생성: API/DB Spec 기반 단위 테스트
	     ⑥ 이슈 대응: 문제 원인 1차 분석 결과 및 조치 가이드 제공
